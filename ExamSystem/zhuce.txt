 在中国，考生只能通过网上报名的方式报考IELTS考试。对于考生来说，仔细阅读下列报名程序及完成相关步骤是非常重要的。

    报名前的准备

电脑要求：你的电脑应与互联网相连接并且装有网络浏览器（最好是IE浏览器）。推荐显示分辨率为1024X768。中国考生还必须提供本人的简体中文姓名和邮寄地址。因此，中国考生必须使用简体中文操作系统来输入汉字。
  
付费须知：你可以使用中国工商银行或中国招商银行的网上付费服务来进行网上付费。这将使你能够通过互联网一次完成整个报名程序，并支付较低的手续费。如果你决定取消报名，你所支付的考试费将会马上被退回到你的银行帐户上。如果你想使用网上付费，但没有上述两家银行的电子商务卡，请到中国工商银行或中国招商银行营业厅去办理电子商务卡。中国工商银行（http://www.icbc.com.cn）和招商银行（http://www.cmbchina.com）在其网站和营业厅均备有相关服务的详细资料。 
所需个人资料：网上报名所需的个人基本资料有：中文姓名，姓名拼音/英文姓名(中国大陆的考生请按新华字典的规则输入标准汉语拼音；香港、澳门和台湾地区的考生请以证件上的拼音或英文姓名为准; 非中国考生直接输入英文姓名)，身份证件类别，身份证件号码，出生日期，电话号码，电子邮箱地址及通信地址。请确保你在上网报名之前拥有上述资料并确认其真实性和准确性。错误的信息将导致你不能得到 IELTS 考试的相关信息及考试成绩。因此，你有责任输入正确和真实的个人信息。你将对输入不正确个人信息造成的后果负责。 
考生规则：在报名之前，你必须阅读IELTS考试考生规则，报名程序及IELTS考试考点信息。要阅读上述信息，你可以点击菜单中的 IELTS信息栏。除此之外，在你填写IELTS报名表之前，你也可以阅读到IELTS考生规则。你必须点击同意相关条款，才能继续进行网上报名操作。
    报名程序

    要进行IELTS考试报名，你必须按顺序完成下列步骤：
    1. 注册成为教育部考试中心网上报名系统用户
    2. 在IELTS考点预定座位
    3. 填写IELTS报名表
    4. 支付IELTS考试费
    只有在完成这四个步骤之后，你的IELTS报名才算完成并得到确认。关于这四个步骤的详细说明如下：

   第一步：注册成为教育部考试中心网上报名系统用户
  
 


    如果你从未使用过教育部考试中心IELTS考试网上报名系统，你必须先注册成为用户。完成这一步，你需要提供基本的个人资料如：姓名、身份证明号码、邮寄地址、电话号码和电子信箱地址。当你提交了所需要的资料之后，你将得到教育部考试中心报名系统分配给你的一个NEEA用户号(NEEA user ID) 号码。在注册之前，你将看到网上报名协议，它为网上报名系统用户设定了相应的条款。你必须点击同意按钮，同意遵守这些条款，否则，系统将不允许你进行注册。当你成功注册成为系统用户后，系统将向你的电子邮箱发送一封确认邮件。 

 重要提示：

    你必须记住 NEEA用户号(NEEA user ID)和密码，这一点非常重要。因此，我们建议你将这些资料存放在一个安全的地方。与他人分享你的NEEA用户号(NEEA user ID)和密码将有可能导致在未经授权的情况下，你个人的IELTS报名信息遭到篡改或损害。 

    你所输入的信息将用于与你本人进行联系及用于其他考试安全方面的用途。

    每次进入教育部考试中心网上报名系统时，你必须输入你的NEEA用户号(NEEA user ID)和密码。只有这样，你才能在你所选择（根据考点的容量）的IELTS考点预定座位，查看你本人的考试费支付状态，填写和提交IELTS报名报名表，确认你的考试日期和考点，或重新安排考试，或取消你的IELTS考试（参照IELTS考试有关规定）。

    你可以使用你的NEEA用户号（NEEA user ID）号码在网上多次报考IELTS考试（根据IELTS考试有关规定）。

    你可以看到你个人的详细资料或通过点击主页上的用户资料按钮来更改你的信息。

 重要提示:

    请注意你的姓名（中文和英文）、性别、身份证明类型、身份证明号码和出生日期，这些在报名系统中的主要信息将被用于 IELTS 考试。你必须确保这些信息是真实和准确的。一但你将这些信息输入系统，系统将不允许你再更改这些信息。

    第二步：在IELTS考点预定座位

当你成功注册成为教育部考试中心网上报名系统的用户之后，系统将引导你到选择IELTS考点的网页上，上面登有所有可供选择参加 IELTS 考试的考点。如果你已注册成为系统用户，但还没有在某一个考点预定过座位，在你成功输入NEEA用户号(NEEA user ID) 和密码之后，系统也将显示同样的考点选择网页。 

    通过选择点击相应的IELTS考点，你将看到上面有报名时间所对应的考试日期的页面及预定座位的按钮。点击预定座位按钮你就可以预定座位。如果无法点击预定座位按钮，则意味着这个考点已经报满或者不举办相应日期的 IELTS考试。请注意：在报名截止日之前，先前已无法点击的预定座位按钮有可能因其它考生取消预定而被重新激活。如果你想预定座位，请经常来此页面查看。

    在点击预定座位按钮之后，就完成了预定座位程序，你将看到“我的状态”这一页面，那里有你目前状态的信息，这些信息包括 IELTS考点的名称，考试日期，你的注册号(Registration ID)，应支付的考试费金额和考试交费的截止日期。系统将给你的电子邮箱发一封准考证。你可以通过点击填写报名表并付费、现在离开并在两天之内回来、取消这三个按钮来选择进入下一步。

    如果点击填写报名表并付费按钮，你将会看到IELTS报名表。当你填完报名表后，系统将提示你支付IELTS考试费。 

    点击现在离开并在两天之内回来填写报名表并付费按钮将使你在预定座位已被确认的情况下离开系统。但是，你应记住必须在两天之内回到网上报名网站来填写IELTS报名表，因为如果在两天之后，你不填写报名表，你所预定的座位将被系统取消。同时，还应记住即使你在两天之内填写了报名表，教育部考试中心还必须在你预定座位的7日内收到你的考试费，否则，你的预定也将被取消。

    如果点击取消按钮，系统将取消你在所选的IELTS考点预定的座位并删除你的报名表。

 重要提示：

    请记录下你的注册号（Registration ID）并把它放在一个安全的地方。注册号（Registration ID）及以后所提供的IELTS考号（IELTS candidate number）都将用来确认你参加考试的资格，并凭此享受IELTS考试相关的服务。

    第三步：填写IELTS报名表

    在我的状态网页上有填写报名表并付款按钮。点击此按钮，你将看到并填写IELTS报名表，你可以通过下列方式进入我的状态网页：1）在成功预定座位后；或2）在成功预定座位后，点击现在离开并在两天之内回来按钮，以退出系统并重新登录教育部考试中心IELTS考试报名网页；3）在任何时候点击菜单栏中的我的状态。

    当你进入IELTS报名表页面前，你必须详细阅读IELTS考试考生规则。在你阅读之后，点击我已了解并同意遵守上述规则旁边的按钮。如你拒绝点击此按钮，你将不能继续报名程序。

当你看到IELTS报名表后，请按照屏幕上的指令进行操作。请注意系统不允许你重复输入你的姓名、性别、生日，身份证件号码，考试日期和考点这些信息，因为在此之前，你已输入过上述信息了。

    当你填写完成IELTS报名表后，点击提交按钮。系统将引导你进入我的状态页面，此页面将显示你目前的状态。然后，你可以选择点击下列三个按钮：编辑IELTS报名表，付费，取消预定。

    点击编辑IELTS报名表按钮将允许你修改你先前所提交的信息。这一功能将在你所选择的考试日期前3周失效。同时系统将不允许你修改你的姓名、性别，出生日期，身份证件号码，考试日期和考点。

    通过点击付费按钮可以支付IELTS考试费。这是IELTS报名程序中所要求的最后一个步骤。

    点击取消预定按钮将取消你所预定的IELTS考试座位并删除你所提交的IELTS报名表。

    第四步：支付IELTS考试费

    你只有通过进入我的状态页面或在完成填写IELTS报名表之后才能实现这一功能。

IELTS费用包括IELTS考试费和IELTS转考费。考生只有在支付IELTS考试费之后才算完成整个报名程序。IELTS转考费是指考生在本人报名之后，报名截止日期之前需要更改所选考点（视情况许可）或考试日期（视情况许可），为此向该考生所收取的费用。

    屏幕上将显示付费的总金额（包括IELTS考试费和IELTS转考费）。

    付费

    目前有三种付费方式：

通过中国工商银行进行网上付费 
通过中国招商银行进行网上付费 
通过任何银行将考费电汇到教育部考试中心的银行帐户上。 
    1.网上付费：通过点击相应的中国工商银行或中国招商银行网上付费按钮便可以开始进行网上付费。你将被引导到中国工商银行或中国招商银行的安全网页。请按照银行的要求完成付费程序。当付费完成后，请记录下银行所提供的交易号码，以便日后用于核对和查询你的付款。通常教育部考试中心IELTS报名系统会立即收到你的付款确认，至此你已完成整个IELTS报名程序。然而，在极少数情况下，银行系统会延迟向教育部考试中心报名系统确认考生的付款。因此，请在24 小时之后，登录教育部考试中心 报名网站，以便核实你的付费和报名状态。如果你仍未看到对你付费的确认，请拨打教育部考试中心IELTS考试服务热线以寻求帮助。如你在银行网页上进行付费时网络连接突然中断，这种情况下，你的网上付费可能没有成功。当教育部考试中心网上报名系统收到银行的付费确认后，系统会给你的电子邮箱发一封准考证邮件。请注意，除了统一的考试费之外，你还需向银行支付服务费，中国工商银行和中国招商银行针对每笔IELTS汇款所收取的服务费分别为人民币6元和5元。

    2.电子汇款付费：当你点击电子汇款按钮，你将看到有关通过电汇方式向教育部考试中心银行帐号支付IELTS费用的重要信息。这些信息包括：教育部考试中心开户银行户名、银行帐号、开户银行名称和地址，以及应汇款的人民币金额。请将电脑屏幕上所显示的上述信息准确地记录下来。请携带上述资料到银行营业厅办理向教育部考试中心电汇考试费的手续。银行将要求你填写一张汇款单，你应确保你所填写的信息是准确无误的。当你仔细核对你所填写的信息之后，请将汇款单和钱一起交给银行的营业员，请你收好银行给你的收据以便将来查询之用。根据银行的具体情况，教育部考试中心的网上报名系统将在 1至4天之内确认收到你的汇款。请你在办理完成汇款至少两天之后登录教育部考试中心的报名网站进行查询。如果你的汇款没有得到确认，请拨打教育部考试中心的呼叫中心来寻求帮助。你有责任将报名网站上所提供的信息准确完整地记录下来。除了统一的IELTS考试费之外，对于通过电汇所支付的每笔IELTS考试费，你还需向教育部考试中心和银行支付相应的手续费，手续费分别为5元人民币和IELTS考试费的1％。

 重要须知：

    电汇并不是为IELTS考试专门开设的一项特殊服务，它只是银行所开设的一项普通汇款服务。因此，当你到银行营业厅时，你必须要求电汇服务。如果你只说要支付IELTS考试费，银行的营业员将不会明白你的要求。

    打印准考证

    当你完成付费程序之后，我的状态页面将显示对你付费的确认，整个IELTS报名程序也随之完成。当教育部考试中心的IELTS网上报名系统收到你的付款之后，将给你的电子邮箱发出一封称作IELTS准考证的邮件，来确认你的报名。你也可以在我的状态栏中看到这封准考证并将它打印下来。

 重要须知：

    你必须将IELTS准考证打印下来，并在考试当天把它带到考点。

    退考－取消报名和退费

    IELTS考试允许考生在报名截止日期之前取消本人先前的报名。这种情况下，将退给考生700元人民币的退考费。你只能在教育部考生中心 IELTS报名网站上取消你的报名。在你准备取消报名之前，先记录下你的注册号(Registration ID)，确认你在个人记录里所提供的邮寄地址准确无误（使用电汇方式付费的考生的退款将通过邮局汇款的方式退给考生；使用网上支付方式付费的考生的退款将被退到考生的银行帐户上）。如你要取消报名，请在进入系统之后点击我的状态按钮以进入我的状态页面。页面上有三个按钮：编辑报名表、转考和取消报名并申请部分退费。点击取消报名并申请部分退费按钮来取消你的报名。在点击取消报名并申请部分退费按钮之后，系统将问你是否进行了误操作。在你确认要取消报名之后，你先前的报名将被取消，700元人民币的退费将被保存在你的个人记录中。考生取消考试后，应当主动与呼叫中心取得联系，办理退款或申请收据寄送等事宜，否则不予办理退款手续。在你与呼叫中心确认后的四周之内，退款将被转到你的银行帐户上或通过邮局汇到你所提供的邮寄地址上。如果你有任何问题或没有收到退款，请给教育部考试中心的呼叫中心打电话。一旦取消了你的报名，你将不能再反悔。如果你想再进行 IELTS报名，你必须从上面所介绍的第二步开始做起。在你取消报名之后，系统将向你的电子信箱发一封通知邮件。

    转考－更改考点及/或考试日期

    执行转考操作

    当你成功完成IELTS考试报名程序之后，在报名截止日期之前，你仍然可以在网上变更你的考试日期和考点，这需要你额外支付500元人民币。当您在选择的考点申请座位时，您会看到每场考试的报名、转考和退考截止日期。请以该网页公布的转考截止日期为准。

    友情提示：

    1. 我的状态

    我的状态页面是了解整个报名过程中每个步骤所处状态的核心。报名之后，将加入一些更重要的信息（IELTS考号，口试时间）。此外，报名之后你仍可以进入我的状态页面来进行下列工作： 1）更改IELTS报名表上的信息（考试日期前3周之内，此功能失效）；2）更改考试日期及/或考点（将额外收取500元人民币，更改必须在距考试日期3周之前进行并取决于实际座位情况）；3）取消报名。在你登录系统之后，你可以通过点击主页上的我的状态按钮以进入取消报名页面。

    2. 关于 注册号（Registration ID）和 IELTS考号（IELTS candidate number）

请注意，在你完成IELTS报名程序之后，教育部考试中心IELTS网上报名网站会分配给你一个注册号（Registration ID）。在IELTS 报名截止日期之后，教育部考试中心IELTS网站将在我的状态网页公布你的IELTS candidate number。Registration ID号将用于从教育部考试中心IELTS 网站和呼叫中心获取相关服务之用。IELTS candidate number号将被用于参加IELTS考试或从英国文化协会得到IELTS考试相关服务之用。

    教育部考试中心IELTS报名系统向考生提供下列号码。他们的名称和用途如下，请了解它们各自的用途和区别：

    NEEA用户号(NEEA user ID)：和系统使用者所设定的密码一起使用，用于保护使用者的信息在未经授权的情况下不被他人看到。

    注册号(Registration ID)：在你完成所有教育部考试中心IELTS 报名系统所要求的报名步骤之后，系统所生成的一个唯一的确认号码。当你给教育部考试中心呼叫中心打电话时，需要提供这个号码。有了这个号码你才能得到IELTS考号(IELTS candidate number)。

    IELTS考号(IELTS candidate number)：在通过教育部考试中心IELTS报名网站得到你的申请数据之后，剑桥大学考试委员会所生成一个特殊的号码。你可以通过此号码获得英国文化协会所提供的IELTS 相关服务，或凭此号码要求寄送你的成绩单给相关机构。
